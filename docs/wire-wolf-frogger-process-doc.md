# Process documentation
FOR THE WIRE WOLF WOODS

## PURPOSE:
The purpose of this procedure is to ensure that all incoming materials are consistently prepared, partitioned, and delivered to the proper bays for long-term integration. This documentation applies whenever a new IMPORT is received on the local grid.

Primary SAM will receive the next material by prophetic imposition. When the event occurs, new IMPORT will be received at the primary local bay. 

## PHASE 1: Preparing the Material

IMPORT the materials from the primary bay. Identify the length and density of the material ahead of any official export.

If the length and/or density of the material exceeds the yard system standards: partition the material into two or more divisions. Use perceived theme & time occultations or use scale for executive decisions on partitioning.

## PHASE 2: Privatizing the Material

REVIEW THE MATERIAL CAREFULLY.
Does it contain sensitive information or structures? Demarcate and redact according to the joint C.N.S. readout.

## PHASE 3: Label and Export

EXPORT the material from the primary local bay in the correct formats for the second external bay import.
USE THE SAME STANDARDS EACH TIME.

### TAG YOUR MATERIAL WITH THE FOLLOWING CONVENTION:
3-digit code for the ORDER NUMBER IN WHICH THE MATERIAL WAS

**RECEIVED BY THE PRIMARY SAM.**

     3-digit code for the ORDER NUMBER IN WHICH THE MATERIAL WAS

**ORDERED WITHIN THE GLASS.**

     1-digit code for the PARTION OF THE ORIGINAL MATERIAL

(in cases that you must break a material down into several imports)

For example: `001.023.3` would imply the material was the first imported to the private bay, and the 23 of the documented order of original import, and the third segment of the original material.

     DO INCLUDE THE SAME TITLE AS THE TITLE USED IN THE ORIGINAL GLASS. If desired by THE C.C., consider adding a second field in the external bay for bonus-naming the materials, especially when breaking down into.

Deliver the final materials to the known drop location. You will be given an KEY for delivery use. THIS KEY WILL BE IMPORTANT.

## PHASE 4: Delivery to the D.B. Facility

The materials are now ready to be delivered to the second external bay. Present your KEY and the TAG data for each segment you will be delivering.

Ideally, please mark the textures and topicals of the material and annotate the details of the material onto the tag, for proper distribution to the Timber and Ark yards.
